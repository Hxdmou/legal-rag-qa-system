# 常见问题

## 🔧 安装与运行

### Q: 如何安装依赖？
**A:** 使用 pip 安装 requirements.txt 中的依赖：
```bash
pip install -r requirements.txt
```

### Q: 系统启动失败怎么办？
**A:** 请检查：
1. Python 版本是否为 3.10+
2. 依赖是否安装完整
3. 端口是否被占用（尝试更换端口）
4. 是否有足够的内存和磁盘空间

### Q: 如何启动单个系统？
**A:** 使用以下命令启动：
```bash
# 通用RAG系统
streamlit run run.py --server.port 7861

# 法律知识系统
streamlit run medical_qa_fixed.py --server.port 7869
```

## 📁 文档处理

### Q: 支持哪些文档格式？
**A:** 支持 PDF、Word、Excel、TXT 等格式。

### Q: 文档上传失败怎么办？
**A:** 请检查：
1. 文件大小是否超过限制
2. 文件格式是否支持
3. 文件是否损坏

### Q: 如何添加自定义知识库？
**A:** 将文档放入 `knowledge_bases/` 目录，然后运行 `build_index.py` 构建索引。

## 🤖 问答功能

### Q: 为什么回答不准确？
**A:** 可能的原因：
1. 知识库中没有相关内容
2. 问题表述不够清晰
3. 需要优化检索参数

### Q: 如何提高问答准确性？
**A:**
1. 添加更多相关文档到知识库
2. 确保问题描述清晰明确
3. 使用更具体的关键词

### Q: 支持多轮对话吗？
**A:** 是的，系统支持多轮对话，会保持对话上下文。

## 💾 数据存储

### Q: 对话记录保存在哪里？
**A:** 对话记录保存在本地 `chat_histories/` 目录下。

### Q: 如何清理历史记录？
**A:** 在系统页面点击"清空对话"或"清空所有历史"按钮，或直接删除 `chat_histories/` 目录。

### Q: 数据会上传到云端吗？
**A:** 不会，所有数据仅在本地处理和存储，保护用户隐私。
