## 📚 项目简介

本项目包含两套智能问答系统，均基于 RAG（Retrieval-Augmented Generation）检索增强生成技术：

| 系统 | 文件 | 端口 | 说明 |
|------|------|------|------|
| **RAG 智能问答系统** | `run.py` | 7861 | 通用版，支持任意文档格式 |
| **法律知识问答系统** | `medical_qa_fixed.py` | 7869 | 法律领域专用版，预置法律知识库 |

---

## 🎯 核心功能

### 通用 RAG 智能问答系统
- ✅ 支持多格式文档（**TXT、PDF、Excel、Word**）
- ✅ 智能文本分块和向量化
- ✅ 混合检索（**BM25 + 向量嵌入**）提升准确性
- ✅ 基于文档内容的精准问答
- ✅ 多轮对话，历史自动保存
- ✅ 对话记录本地存储

### 法律知识问答系统
- ✅ 预置丰富法律知识库（法律法规、司法解释、典型案例）
- ✅ 专业法律问答，引用法条依据
- ✅ 智能法律文书解读
- ✅ 多轮对话，深入分析法律问题
- ✅ 知识库可视化管理

## 🚀 快速开始

### 环境要求
- Python 3.10+
- pip 包管理器

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动系统

#### 启动通用 RAG 智能问答系统
```bash
streamlit run run.py --server.port 7861
```
访问地址：`http://localhost:7861`

#### 启动法律知识问答系统
```bash
streamlit run medical_qa_fixed.py --server.port 7869
```
访问地址：`http://localhost:7869`

#### 一键启动（Windows）
双击运行 `启动RAG系统.bat` 即可同时启动两个系统，并自动打开浏览器。

## 💡 使用说明

### 上传文档
1. 打开系统网页后，在左侧sidebar找到「上传文档」区域
2. 支持拖拽或点击上传文件
3. 支持格式：`.txt`、`.pdf`、`.xlsx`、`.docx`
4. 系统会自动进行文本分块和向量化处理

### 智能问答
1. 在底部输入框输入您的问题
2. 系统会在已上传的文档中检索相关内容
3. 基于检索结果生成准确答案
4. 支持多轮对话，可以追问深入探讨

### 一键启动
- 双击 `启动RAG系统.bat`
- 等待约5秒，浏览器自动打开两个系统页面

## 🔧 技术栈

| 类别 | 技术 |
|------|------|
