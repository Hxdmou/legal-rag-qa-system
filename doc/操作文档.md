## 三、数据准备方式

### 1. 数据下载

1. 注册 Kaggle 账号：https://www.kaggle.com/
2. 进入竞赛页面：[Cat or Dog: Image Classification Challenge](https://www.kaggle.com/c/dogs-vs-cats)
3. 下载训练集和测试集数据

| 类别 | 训练集数量 | 验证集数量（自动划分） |

数据划分比例：训练集:验证集 = 8:2

## 四、训练命令与流程

### 1. 训练命令

```bash
# 使用 ResNet18 训练（默认配置）
python train.py --data_dir ./data --epochs 15 --batch_size 32 --model resnet18

# 使用 VGG16 训练（冻结特征提取层）
python train.py --data_dir ./data --epochs 20 --batch_size 16 --model vgg16 --freeze

# 使用 ResNet50 训练（自定义学习率）
python train.py --data_dir ./data --epochs 10 --batch_size 16 --model resnet50 --lr 0.0001
```

### 2. 训练参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| --data_dir | ./data | 数据集目录 |
| --epochs | 15 | 训练轮数 |
| --batch_size | 32 | 批大小 |
| --lr | 0.001 | 学习率 |
| --model | resnet18 | 模型类型（alexnet/vgg16/vgg19/resnet18/resnet34/resnet50/googlenet） |
| --freeze | False | 是否冻结预训练层 |
| --save_dir | ./results/models | 模型保存目录 |
| --figure_dir | ./results/figures | 图表保存目录 |

## 五、模型测试/推理使用方法

### 1. 单张图片预测

```bash
python inference.py --model_path ./results/models/best_resnet18.pth \
                    --img_path ./data/test/cat.1.jpg \
                    --model resnet18
```

输出示例：
```
Model loaded from ./results/models/best_resnet18.pth
Image: ./data/test/cat.1.jpg
Prediction: cat (Confidence: 0.9876)
Cat Probability: 0.9876, Dog Probability: 0.0124
```

### 2. 批量图片预测

```bash
python inference.py --model_path ./results/models/best_resnet18.pth \
                    --img_path ./data/test \
                    --model resnet18 \
                    --output_csv ./results/predictions.csv
```

### 3. 推理参数说明

| 参数 | 说明 |
|------|------|
| --model_path | 必需，训练好的模型路径 |
| --img_path | 必需，图片文件或目录路径 |
| --model | 模型类型，需与训练时一致 |
| --output_csv | 可选，输出CSV文件路径 |

## 六、预期结果与常见问题

### 1. 预期准确率

| 模型 | 验证集准确率范围 |
|------|----------------|
| AlexNet | 85% - 90% |
| VGG16/VGG19 | 90% - 95% |
| ResNet18 | 92% - 96% |
| ResNet50 | 94% - 97% |

### 2. 常见问题与解决方案

#### 问题1：显存不足

**现象**: `RuntimeError: CUDA out of memory`

**解决方案**:
- 减小 batch_size（如从 32 改为 16）
- 使用更小的模型（如 ResNet18 替代 ResNet50）
- 冻结预训练层（添加 --freeze 参数）

#### 问题2：训练数据找不到

**现象**: `FileNotFoundError: [Errno 2] No such file or directory: './data/train'`

**解决方案**:
- 确认数据已正确下载并解压
- 检查目录结构是否正确
- 使用 --data_dir 参数指定正确路径

#### 问题3：模型文件损坏

**现象**: `RuntimeError: Error(s) in loading state_dict for ResNet`

**解决方案**:
- 确认模型文件路径正确
- 确认推理时 --model 参数与训练时一致
- 重新训练模型

#### 问题4：图片读取错误

**现象**: `PIL.UnidentifiedImageError: cannot identify image file`

**解决方案**:
- 检查图片文件是否损坏
- 确保只包含 png/jpg/jpeg 格式图片
- 在 inference.py 中添加异常处理
