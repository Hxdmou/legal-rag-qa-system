## 🎯 技术亮点

1. **统一架构**: 所有系统基于同一套RAG框架构建，保证稳定性和一致性
2. **私有化部署**: 支持本地部署，数据安全可控
3. **多模型支持**: 支持阿里云百炼、OpenAI等多种大模型
4. **智能索引**: 预置知识库一键加载，开箱即用
5. **灵活配置**: 支持温度、检索数量、检索模式等参数自定义
