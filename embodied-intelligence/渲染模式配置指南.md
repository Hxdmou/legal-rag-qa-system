# Isaac Sim 渲染模式配置指南

## 渲染模式选择

在创建 `SimulationApp` 时，通过 `renderer` 参数选择渲染模式：

```python
from isaacsim import SimulationApp

simulation_app = SimulationApp({"renderer": "RealTimePathTracing"})
```

## 可用渲染模式

### 1. RealTimePathTracing (推荐)
- **描述**: RTX Real-Time 2.0
- **适用场景**: 大多数交互式机器人和仿真工作流程
- **特点**: 质量与性能最佳平衡
- **推荐指数**: ⭐⭐⭐⭐⭐

```python
simulation_app = SimulationApp({"renderer": "RealTimePathTracing"})
```

### 2. PathTracing
- **描述**: RTX Interactive Path Tracing
- **适用场景**: 最终捕获、质量比较和验证图像
- **特点**: 高质量路径追踪，画质优先于帧时间
- **推荐指数**: ⭐⭐⭐⭐

```python
simulation_app = SimulationApp({"renderer": "PathTracing"})
```

### 3. RaytracedLighting
- **描述**: RTX Real-Time (Legacy)
- **适用场景**: 兼容旧项目
- **特点**: 传统实时光线追踪
- **推荐指数**: ⭐⭐⭐

```python
simulation_app = SimulationApp({"renderer": "RaytracedLighting"})
```

### 4. MinimalRendering
- **描述**: RTX Minimal
- **适用场景**: 高吞吐量/低延迟关键应用
- **特点**:
  - 禁用间接光传输
  - 仅使用场景中第一个光源
  - 仅支持硬阴影
  - 无法使用全部材质效果
- **推荐指数**: ⭐⭐⭐

```python
simulation_app = SimulationApp({"renderer": "MinimalRendering"})
```

### 5. Minimal
- **描述**: No Rendering
- **适用场景**: 计算机视觉任务（深度、法线、分割等）
- **特点**: 黑色输出，仅非颜色 AOV
- **推荐指数**: ⭐⭐

```python
simulation_app = SimulationApp({"renderer": "Minimal"})
```

## RTX-Minimal 详细配置

当使用 `MinimalRendering` 时，可以通过以下参数配置：

### 着色模式 (minimal_shading_mode)

```python
import carb.settings

settings = carb.settings.get_settings()

# 模式 0: 实时2.0（参考文献）
settings.set("/rtx/minimal/mode", 0)

# 模式 1: 漫射/光泽/发射
settings.set("/rtx/minimal/mode", 1)

# 模式 2: 纹理漫射
settings.set("/rtx/minimal/mode", 2)

# 模式 3: 恒定弥漫
settings.set("/rtx/minimal/mode", 3)

# 模式 4: 无渲染（黑色输出）
settings.set("/rtx/minimal/mode", 4)
```

### 环境光设置

```python
# 环境光颜色 (RGB)
settings.set("/rtx/sceneDb/ambientLightColor", [0.5, 0.5, 0.5])

# 环境光强度
settings.set("/rtx/sceneDb/ambientLightIntensity", 1.0)
```

### 常数颜色

```python
settings.set("/rtx/minimal/constantColor", [1.0, 1.0, 1.0])
```

## 运行时更改渲染模式

在 Isaac Sim 运行时，可以通过 `carb.settings` 更改渲染模式：

```python
import carb.settings

# 运行时切换渲染器
settings = carb.settings.get_settings()
settings.set("/rtx/rendermode", "PathTracing")
```

## 半透明材质设置

对于需要正确渲染半透明/切割材质的场景（如分层复合材料、织物、玻璃、液体等）：

```python
import carb.settings

settings = carb.settings.get_settings()
settings.set("/rtx/pathtracing/fractionalCutoutOpacity", True)
```

**注意**: 此设置会增加渲染成本，如果工作流程不需要半透明效果，可禁用以提升性能。

## 场景推荐配置

| 场景 | 推荐渲染器 | 原因 |
|------|----------|------|
| 机器人仿真/控制 | RealTimePathTracing | 质量与性能最佳平衡 |
| 交互式开发/调试 | RealTimePathTracing | 实时反馈 |
| 最终渲染/质量验证 | PathTracing | 高质量路径追踪 |
| 批量仿真/数据生成 | MinimalRendering | 高吞吐量 |
| 计算机视觉(深度/分割) | Minimal | 仅需非颜色AOV |
| 延迟关键应用 | MinimalRendering | 最小延迟 |

## 完整示例

```python
import os
os.environ["OMNI_KIT_ACCEPT_EULA"] = "1"

from isaacsim import SimulationApp
import carb.settings

# 选择渲染模式
RENDERER_MODE = "RealTimePathTracing"  # 机器人仿真推荐

# 创建 SimulationApp
simulation_app = SimulationApp({
    "headless": False,  # True=无窗口，False=显示图形界面
    "renderer": RENDERER_MODE
})

# 可选：配置半透明材质（如果需要）
settings = carb.settings.get_settings()
settings.set("/rtx/pathtracing/fractionalCutoutOpacity", True)

# 如果使用 MinimalRendering，可设置着色模式
if RENDERER_MODE == "MinimalRendering":
    settings.set("/rtx/minimal/mode", 1)  # 漫射/光泽/发射
    settings.set("/rtx/sceneDb/ambientLightColor", [0.5, 0.5, 0.5])
    settings.set("/rtx/sceneDb/ambientLightIntensity", 1.0)

# 验证当前渲染模式
current_renderer = settings.get("/rtx/rendermode")
print(f"当前渲染器: {current_renderer}")

# 运行仿真...
# simulation_app.close()
```

## 渲染设置优化

### 减少 GPU 内存使用

```python
settings = carb.settings.get_settings()

# 减少纹理流媒体预算
# 默认: 0.6 (GPU 内存的 60%)
settings.set("/rtx-transient/resourcemanager/texturestreaming/memoryBudget", 0.3)
```

### 减少潜在内存泄漏（实验性）

```python
import os
os.environ["GLIBC_TUNABLES"] = "glibc.malloc.arena_max=1:glibc.malloc.mmap_max=0:glibc.malloc.mmap_threshold=2147483647"
```

此设置有助于在长时间运行的工作流中减少内存泄漏。
